Proje Hakkında

Bu proje, Espark’ta faaliyet gösteren Kartur Lojistik firması için geliştirilmiştir. Uygulama yalnızca backend katmanını içermekte olup tüm mevcut kodlar tarafımca geliştirilmiştir.

Özellikler

Backend odaklı mimari

Mevcut tüm servisler aktif ve çalışır durumdadır

API testleri için Swagger UI entegre edilmiştir

Well-formed ve sürdürülebilir kod yapısı

Local veritabanı kullanımı

Kullanım

Proje, arayüz (frontend) geliştirilerek doğrudan kullanılabilir. Mevcut API yapısı frontend entegrasyonuna uygundur.

Test

API uç noktaları Swagger UI üzerinden test edilebilir.
